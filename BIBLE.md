# 📖 BIBLE — Drôles de Droits

> **La bible créative complète** du jeu narratif interactif sur les droits des jeunes en centre jeunesse au Québec.
> 
> *Dernière mise à jour : Décembre 2025*

---

## 🎯 Vue d'ensemble

### Concept
**Drôles de Droits** est un jeu narratif interactif de type *visual novel* destiné aux adolescents placés en centres jeunesse au Québec. À travers des histoires immersives aux choix multiples, les jeunes découvrent leurs **12 droits d'usagers** du système de santé et des services sociaux de manière engageante et mémorable.

### Mission
> *"Transformer des droits abstraits en expériences vécues"*

Plutôt que de présenter des droits sous forme de listes ou de documents juridiques, le jeu plonge le joueur dans des situations réalistes où ces droits deviennent des outils concrets pour naviguer des situations difficiles.

### Public cible
- **Âge** : 12-18 ans
- **Contexte** : Jeunes hébergés en centres jeunesse, ressources intermédiaires, familles d'accueil
- **Utilisation** : Individuel ou accompagné (éducateur, intervenant DPJ)

### Ton et approche
- **Réaliste mais accessible** : Situations crédibles sans être traumatisantes
- **Empowerment** : Le joueur découvre qu'il a du pouvoir via ses droits
- **Non-moralisateur** : Pas de leçon condescendante, le joueur apprend par l'expérience
- **Québécois authentique** : Langue, décors, contexte 100% local

## 🏛️ Les Lieux

### Centre Jeunesse (Intérieur)

#### 🚪 Couloir Principal
- **Type** : Intérieur institutionnel
- **Ambiance** : Point de passage central, tentative de rendre l'espace accueillant
- **Détails visuels** :
  - Murs bicolores : bleu pâle en haut, orange brûlé en bas
  - Sol en linoléum gris-beige usé mais propre
  - Néons fluorescents + fenêtres pour lumière naturelle
  - Babillards avec affiches (droits, activités, règlements)
  - Dessins de jeunes, photos d'événements
- **Sons** : Échos de voix, portes, parfois musique lointaine
- **Variations** : Jour (actif), soir (calme), nuit (sécurité réduite)

#### 🍽️ Cafétéria
- **Type** : Grande salle commune
- **Ambiance** : Bruyante aux repas, lieu de socialisation clé
- **Détails visuels** :
  - Fresques murales colorées faites par les jeunes
  - Tables rectangulaires, chaises plastique multicolores
  - Grandes fenêtres sur un mur
  - Menu du jour sur tableau
- **Sons** : Brouhaha, couverts, bruits de plateau
- **Fonction narrative** : Lieu où les conversations importantes se passent

#### 🛏️ Chambre d'Alex
- **Type** : Chambre individuelle
- **Ambiance** : Intime, personnalisée dans les limites permises
- **Détails visuels** :
  - Murs blancs/beige avec posters autorisés
  - Lit simple, bureau, petite armoire
  - Fenêtre avec barreaux discrets
  - Photos de souvenirs, quelques objets personnels
- **Variations** : Jour (ordinaire), soir (cozy), insomnie (lumière de téléphone)
- **Fonction narrative** : Refuge ET prison selon l'humeur

#### 💼 Bureau de Sophie
- **Type** : Bureau d'intervenante
- **Ambiance** : Professionnel mais chaleureux, sécurisant
- **Détails visuels** :
  - Couleur chaude (beige/pêche)
  - Diplômes encadrés
  - Plantes, photos inspirantes
  - Deux chaises confortables pour les visiteurs
  - Boîte de mouchoirs visible
- **Sons** : Calme, conversation feutrée
- **Fonction narrative** : Espace de confiance pour dialogues importants

#### 🛋️ Salle Commune
- **Type** : Espace détente
- **Ambiance** : Détendue, parfois bruyante
- **Détails visuels** :
  - Fresques colorées
  - Sofas usés mais confortables
  - TV grand écran, baby-foot
  - Poufs, jeux de société
- **Sons** : TV, rires, jeux vidéo

#### 🏥 Infirmerie
- **Type** : Local médical
- **Ambiance** : Clinique, propre, odeur de désinfectant
- **Fonction narrative** : Droit aux soins, distribution de médicaments

#### 📚 École du Centre
- **Type** : Salle de classe (6-8 élèves max)
- **Ambiance** : Scolaire mais plus personnalisée qu'une école normale

### Centre Jeunesse (Extérieur)

#### 🏢 Vue Extérieure du Centre
- **Architecture** : Briques rouges/beiges, 2-3 étages, style années 70-80 rénové
- **Éléments** : Portes vitrées, rampe d'accès, petit stationnement, bancs
- **Variations météo** :
  - Été : Verdure, jeunes dehors
  - Hiver : Neige, québécois typique
  - Pluie : Mélancolique, reflets sur l'asphalte
  - Nuit : Éclairage sécuritaire, ambiance isolée

### Extérieur (Danger)

#### 🌃 Ruelle / Squat Abandonné
- **Type** : Montréal urbain, zone dangereuse
- **Ambiance** : Froide, inhospitalière, CONTRASTE avec le centre
- **Détails visuels** :
  - Ruelle entre bâtiments industriels
  - Bennes à ordures, graffitis
  - Asphalte craquelé, flaques, déchets
  - Cartons, matelas sale, feu improvisé
- **Dangers** : Froid, faim, insalubrité, dealers, criminalité
- **Sons** : Circulation lointaine, sirènes, vent, rats
- **Fonction narrative** : Montrer la réalité de la rue vs sécurité du centre

### Officiel

#### ⚖️ Tribunal de la Jeunesse
- **Type** : Salle d'audience
- **Ambiance** : Solennelle, intimidante
- **Détails** : Boiseries, sceau du Québec, drapeau, banc surélevé du juge
- **Fonction narrative** : Moments critiques, décisions importantes

- **3 plans** : Premier plan, sujet, arrière-plan
- **Types de plans** :
  - `establishing` : Plan large, lieu entier

❌ Images trop sombres/effrayantes
❌ Violence ou drogue graphique
❌ Texte en anglais
❌ Style anime/manga japonais
❌ Proportions irréalistes

## ⚖️ Les 12 Droits des Usagers

### Vue d'ensemble
Basés sur la **Loi sur les services de santé et les services sociaux (LSSSS)** du Québec.

| # | Droit | Icône | Couleur |
|---|-------|-------|---------|
| 1 | Droit à l'information | 📋 | Violet |
| 2 | Droit aux services | 🏥 | Bleu |
| 3 | Droit de choisir son professionnel | 👨‍⚕️ | Vert |
| 4 | Droit de recevoir les soins requis | ⭐ | Jaune |
| 5 | Droit de consentir ou refuser | ✋ | Rouge |
| 6 | Droit de participer aux décisions | 🤝 | Orange |
| 7 | Droit d'être accompagné | 👥 | Teal |
| 8 | Droit à l'hébergement | 🏠 | Indigo |
| 9 | Droit à la langue anglaise | 🗣️ | Lime |
| 10 | Droit au dossier d'usager | 📁 | Cyan |
| 11 | Droit à la confidentialité | 🔒 | Rose |
| 12 | Droit de porter plainte | 📢 | Rose foncé |

### Détails par Droit

#### 1. 📋 Droit à l'information
> *"Tu as le droit d'être informé de manière claire sur ton état de santé, les traitements proposés, leurs risques et bénéfices."*

**Scénarios typiques** :
- Plan d'intervention avec jargon incompréhensible
- Nouveau médicament sans explications
- Évaluation psychologique mystérieuse

#### 2. 🏥 Droit aux services
> *"Tu as le droit de recevoir des services appropriés, continus, sécuritaires et de qualité."*

**Scénarios typiques** :
- Liste d'attente trop longue pour le psy
- Changement d'éducateur sans transition
- Crise d'anxiété minimisée

#### 3. 👨‍⚕️ Droit de choisir son professionnel
> *"Tu peux choisir le professionnel qui te soigne et l'établissement où tu veux recevoir des soins."*

**Scénarios typiques** :
- Pas de feeling avec le psychologue assigné
- Préférence pour un spécialiste externe
- Malaise avec un médecin pour sujets intimes

#### 4. ⭐ Droit de recevoir les soins requis
> *"Tu as le droit de recevoir des soins professionnels, sécuritaires et adaptés à ton état."*

**Scénarios typiques** :
- Migraines chroniques ignorées
- Besoins de santé mentale non adressés

#### 5. ✋ Droit de consentir ou refuser
> *"Personne ne peut te forcer à recevoir des soins contre ta volonté (sauf urgence extrême)."*

**Scénarios typiques** :
- Pression pour prendre un médicament
- Refus d'un traitement controversé

#### 6. 🤝 Droit de participer aux décisions
> *"Tu as le droit d'être impliqué dans les décisions concernant tes soins."*

**Scénarios typiques** :
- Plan d'intervention décidé sans toi
- Objectifs imposés sans consultation

#### 7. 👥 Droit d'être accompagné
> *"Tu peux être accompagné par une personne de confiance lors de tes rendez-vous."*

**Scénarios typiques** :
- Rendez-vous stressant avec le psychiatre
- Besoin de soutien moral

#### 8. 🏠 Droit à l'hébergement
> *"Si ton état le requiert, tu as le droit d'être hébergé dans un établissement adapté à tes besoins."*

**Scénarios typiques** :
- Unité inadaptée à tes besoins
- Demande de transfert refusée

#### 9. 🗣️ Droit à la langue anglaise
> *"Si tu es anglophone, tu as le droit de recevoir des services en anglais dans les établissements reconnus."*

#### 10. 📁 Droit au dossier d'usager
> *"Tu as le droit de consulter ton dossier médical et d'en obtenir une copie."*

**Scénarios typiques** :
- Curiosité sur ce qui est écrit dans ton dossier DPJ
- Demande d'accès refusée

#### 11. 🔒 Droit à la confidentialité
> *"Tes informations de santé sont confidentielles et ne peuvent être partagées qu'avec ton consentement."*

**Scénarios typiques** :
- Éducateur qui parle de toi en pause café
- Informations personnelles qui circulent

#### 12. 📢 Droit de porter plainte
> *"Si un de tes droits n'a pas été respecté, tu peux porter plainte et être assisté."*

**Scénarios typiques** :
- Traitement irrespectueux d'un éducateur
- Peur des représailles après une plainte

## 📚 Structure des Chapitres

### Format d'un Chapitre

Chaque chapitre est un fichier JSON autonome suivant cette structure :

```
{
  metadata: {
    id, slug, title, subtitle,
    description, right (droit associé),
    characters, estimatedDuration,
    difficulty, tags, status
  },
  dialogue: {
    intro: [...lignes],
    branche_a: [...lignes],
    branche_b: [...lignes],
    branche_c: [...lignes],
    fin_positive: [...lignes],
    fin_negative: [...lignes]
  },
  choiceMappings: [...],
  unlockableRights: [...],
  images: [...],
  config: { startScene, endScreens }
}
```

### Anatomie d'une Ligne de Dialogue

```typescript
{
  speaker: "alex" | "marco" | "narrateur" | ...,
  text: "Dialogue en français québécois",
  image: "/images/chapitres/chapitre/scene_XX.webp",
  emotion: "nerveux" | "confiant" | "triste" | ...,
  choices?: ["Option 1", "Option 2", "Option 3"],
  audioFile?: "/audio/alex/alex_scene_01.mp3"
}
```

### Émotions Disponibles

| Émotion | Usage |
|---------|-------|
| `neutre` | Dialogue standard |
| `nerveux` | Stress, inquiétude |
| `confiant` | Assurance, détermination |
| `triste` | Peine, déception |
| `en_colere` | Frustration, rage |
| `peur` | Terreur, anxiété aiguë |
| `determine` | Résolution ferme |
| `hesitant` | Doute, indécision |
| `soulage` | Après une tension résolue |
| `choque` | Surprise négative |
| `frustre` | Irritation contenue |
| `complice` | Complicité, clin d'œil |
| `tension` | Moment de choix crucial |
| `victoire` | Succès, accomplissement |
| `lecon` | Moment de réalisation |
| `info` | Transmission d'information |
| `decouverte` | Apprentissage d'un droit |

```
│   └── images/           # Assets visuels
├── types/                # Définitions TypeScript
└── hooks/                # Hooks React personnalisés
```

### Flux de Données d'un Chapitre

```
1. Chargement du JSON du chapitre
2. Affichage écran d'intro (titre, description, droit)
3. Boucle de jeu:
   a. Afficher image de scène
   b. Afficher bulle de dialogue
   c. Jouer voice-over (si disponible)
   d. Si choix → afficher options
   e. Navigation vers scène suivante selon choiceMapping
4. Vérification des droits à débloquer
5. Écran de fin (positif ou négatif)
```
